# HR
This project focuses on analyzing HR data using SQL for data extraction and Power BI for building an interactive dashboard to support HR decision-making.

🔍 What I Did:
🗄 Data Extraction & Analysis (SQL)
Queried HR dataset using SQL to explore employee data
Applied:
GROUP BY for aggregations
WHERE for filtering (e.g., departments, job roles)
ORDER BY for ranking insights
Calculated key HR metrics such as:
Total Employees
Attrition Rate
Average Salary
Employee Distribution by Department
🔄 Data Cleaning & Preparation
Handled missing values and inconsistent records
Standardized columns (e.g., job roles, departments, dates)
Ensured data quality before loading into Power BI
📊 Data Visualization (Power BI)
Built a fully interactive HR dashboard
Designed visuals including:
Employee distribution by department and role
Attrition analysis
Salary insights
Hiring trends over time
Added slicers for dynamic filtering (e.g., department, gender, experience)
📈 Key Insights:
Identified departments with high attrition rates
Analyzed workforce distribution across roles
Highlighted salary patterns and potential gaps
Provided insights to improve employee retention
🛠 Tools Used:
SQL Server
Power BI
💡 Project Objective:

To analyze HR data and provide actionable insights that help improve:

Employee retention
Workforce planning
HR decision-making
📌 Business Recommendations:
Focus on departments with high attrition and investigate root causes
Adjust salary structures if inconsistencies are found
Improve hiring strategy based on trends and workforce needs
